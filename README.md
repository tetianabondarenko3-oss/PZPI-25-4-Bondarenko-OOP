# Програма «Магазин»

Курсова робота з дисципліни «Об'єктно-орієнтоване програмування».  
Харківський національний університет радіоелектроніки, кафедра ПІ, 2026.  
Виконала: Бондаренко Тетяна Олександрівна ПЗПІ-25-4

## Завдання

Розробити настільний додаток на мові C# з графічним інтерфейсом користувача
для автоматизації роботи магазину. Програма має забезпечувати облік товарів,
реєстрацію продажів, надходжень, повернень, уцінок та списань,
а також формування звітів.

## Функціональність

- Аутентифікація (ролі: Адміністратор, Касир)
- Управління товарами
- Реєстрація надходжень від постачальників
- Оформлення продажів і чеків
- Оформлення повернень
- Уцінка та списання товарів
- Інвентаризація залишків
- Аналітичні звіти
- Управління обліковими записами

## Запуск

1. Завантажте архів з розділу [Releases](../../releases)
2. Розпакуйте у будь-яку папку
3. Запустіть `shop.exe`

> Потрібен [.NET 10.0 Runtime](https://dotnet.microsoft.com/download/dotnet/10.0)
